Client-Side Architecture

How this tool protects your privacy using WebAssembly.

Privacy Standards

Unlike traditional online converters, this tool never uploads your images to an external server. Everything happens strictly inside your browser memory.

WebAssembly

We use Pyodide, a port of CPython to WebAssembly. This enables us to run the Pillow (PIL) library directly in Chrome, Firefox, or Safari.

APIs Used

APIs used in the program is DOM, Pillow, and Canvas 2D Drawing API.
